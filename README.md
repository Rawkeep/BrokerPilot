# BrokerPilot

Der ultimative KI-Copilot für Broker, Banker und Investoren (inkl. Krypto).

## Vision

Ein Multi-Agent-Dashboard das Brokern, Bankern und Investment-Beratern den gesamten Workflow automatisiert — von Lead-Generierung über Marktanalyse bis zum Deal-Abschluss.

## Broker-Typen

- Immobilienmakler — Objektanalyse, Exposés, Marktvergleiche
- Versicherungsmakler — Tarifvergleiche, Bedarfsanalyse
- Finanzberater/Banker — Portfolio-Management, Anlagestrategien
- Krypto-Berater — Token-Analyse, DeFi-Strategien, Whale-Tracking
- Investment-Banker — M&A-Analyse, Due Diligence, Deal-Flow

## Tech-Stack

- **Frontend:** React 18 + Vite 6
- **Backend:** Express.js + LangChain Agent-Orchestrierung
- **AI:** BYOK Multi-Provider (Claude, GPT, Gemini, Mistral, Groq, OpenRouter)
- **Context:** context-mode MCP-Server für Session-Persistenz
- **APIs:** Kuratierte Finanz-, Trading-, Krypto- und Lead-Gen-APIs
- **Deployment:** Docker + Railway

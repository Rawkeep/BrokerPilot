| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-04 | Tampering | LeadForm inputs | mitigate | validateLead called before persist. Name max 200 chars, notes max 5000 chars. React escapes all rendered values. |
| T-02-05 | Tampering | DnD stage assignment | mitigate | handleDragEnd validates newStage exists in BROKER_TYPES[brokerType].pipelineStages before calling moveLead |
| T-02-06 | Information Disclosure | Lead data in DOM | accept | Client-side app, user's own data. No sensitive data leakage concern. |
</threat_model>

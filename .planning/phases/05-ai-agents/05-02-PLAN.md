    **`client/src/stores/agentStore.js`** — Zustand store with localStorage persist:
    - State: `runs` array of `{runId, agentType, leadId, status ('idle'|'enriching'|'thinking'|'done'|'error'), result, error, createdAt, duration}`, `activeRunId` string|null.
    - Actions: `startRun(runId, agentType, leadId)`, `updateRunStatus(runId, status, data?)`, `setRunResult(runId, result)`, `setRunError(runId, message)`, `completeRun(runId, duration)`.
    - Selectors: `getRunsByLeadId(leadId)`, `getActiveRun()`, `getRunsByAgentType(agentType)`.
    - Persist only `runs` array (not activeRunId).

    **`client/src/hooks/useAgentStream.js`** — Custom hook:
    - `useAgentStream()` returns `{ triggerAgent, isStreaming, currentStep, error }`.
    - `triggerAgent(agentType, payload)` function: reads `selectedProvider`, `selectedModel` from `useAIStore`, reads BYOK key from `useKeyStore` (same pattern as useAIRequest.js). Calls `fetch('/api/agents/run', { method: 'POST', headers: {'Content-Type': 'application/json'}, body: JSON.stringify({agentType, payload, provider, model, apiKey}) })`.
    - **Parse SSE from fetch response** (NOT EventSource — POST body not supported by EventSource): Read response body as ReadableStream, use `getReader()` + TextDecoder. Parse SSE lines: split by `\n\n`, extract `event:` and `data:` fields.
    - On each SSE event, update `agentStore` accordingly: `agent:start` -> `startRun`, `agent:enriching` -> `updateRunStatus('enriching')`, `agent:thinking` -> `updateRunStatus('thinking')`, `agent:result` -> `setRunResult`, `agent:error` -> `setRunError`, `agent:done` -> `completeRun`.
    - Track `currentStep` state: `null | 'starting' | 'enriching' | 'thinking' | 'done' | 'error'`.
    - Clean up on unmount (abort controller).

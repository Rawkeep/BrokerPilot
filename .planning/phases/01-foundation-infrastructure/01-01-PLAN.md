       ```dockerfile
       # Stage 1: Build Frontend
       FROM node:22-alpine AS builder
       WORKDIR /app
       COPY package.json package-lock.json ./
       RUN npm ci
       COPY . .
       RUN npm run build

       # Stage 2: Production Server
       FROM node:22-alpine AS production
       WORKDIR /app
       ENV NODE_ENV=production
       COPY package.json package-lock.json ./
       RUN npm ci --omit=dev
       COPY server/ ./server/
       COPY shared/ ./shared/
       COPY --from=builder /app/dist ./dist
       EXPOSE 3000
       USER node
       CMD ["node", "server/index.js"]
       ```

    2. Create `docker-compose.yml` for local development:
       ```yaml
       version: '3.8'
       ```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Spoofing | Express /api routes | mitigate | Rate limiting via express-rate-limit (100 req/15min) on all /api routes |
| T-01-02 | Tampering | HTTP headers | mitigate | Helmet middleware sets CSP, HSTS, X-Frame-Options, X-Content-Type-Options |
| T-01-03 | Denial of Service | Express server | mitigate | Rate limiting + JSON body parser (default 100kb limit) |
| T-01-04 | Information Disclosure | Error responses | mitigate | Error handler returns generic message, stack trace only in non-production |
</threat_model>

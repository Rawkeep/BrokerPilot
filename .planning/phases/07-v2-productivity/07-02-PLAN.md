    - Props: `{ brokerType }` (optional — if omitted, exports all leads)

    - On click: calls `exportLeadsToCSV(filteredLeads, brokerType)` then `downloadCSV(csv, filename)`

    Create `client/src/components/export/ImportModal.jsx`:
    - Props: `{ isOpen, onClose, brokerType }`
    - Step 1 — File selection:
      - File input accepting .csv files
      - On file select: `parseCSVFile(file)` → store parsed data
      - Show preview table (first 5 rows)
    - Step 2 — Column mapping:
      - Auto-detect columns with `autoDetectColumns(headers)`
      - Show a mapping UI: each CSV column → dropdown of lead fields
      - User can adjust mappings
    - Step 3 — Import:
      - Call `mapCSVToLeads(rows, mapping, brokerType)` → get valid/invalid split
      - Show summary: "{N} Leads werden importiert, {M} uebersprungen"
      - Show invalid rows with error messages
      - "Importieren" button: for each valid lead, call `leadStore.addLead(brokerType, leadData)`
      - On complete: show success message, close modal
    - Limit: max 1000 rows (show warning if file has more)
    - Use glass-styled modal/dialog consistent with existing UI patterns

    - Props: `{ agentType, result, metadata }` where metadata = { leadName, brokerType, date }
    - On click: calls `downloadReport(agentType, result, metadata)`

| T-07-07 | Information Disclosure | CSV export | accept | Exports user's own data from their browser; no server data exposure |
